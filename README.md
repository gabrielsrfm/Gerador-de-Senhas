# Gerador de Senhas

Projeto desenvolvido com HTML, CSS e JavaScript puro.  
Permite criar senhas aleatórias e seguras de forma simples e rápida.

## Funcionalidades

- Escolha o comprimento da senha (6 a 32 caracteres)
- Opções para incluir maiúsculas, minúsculas, números e símbolos
- Indicador visual de força da senha
- Botão para copiar a senha com um clique

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Selecione as opções desejadas e clique em **Gerar Senha**
